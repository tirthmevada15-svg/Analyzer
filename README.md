PROJECT OVERVIEW
This Excel workbook is a complete data analysis project created for academic / final-year submission.
It demonstrates practical use of Excel formulas, Data Analysis Toolpak, Pivot Tables, Charts, and Dashboarding.
The project shows an end-to-end data analysis workflow from raw data to business insights.

---

WORKBOOK STRUCTURE (SHEET-WISE EXPLANATION)

1. Raw_Data
   This sheet contains the original dataset.
   Columns included:

* Customer_ID
* Customer_Name
* Region
* Product_Category
* Sales
* Quantity
* Discount
* Order_Date
* Profit

Note: This sheet should not be edited manually.

---

2. Data_Cleaning
   This sheet is used for preparing the data for analysis.
   Includes:

* Month extraction from Order_Date
* Timestamp column using NOW()
* Helper and calculated fields if required

---

3. High_Value_Customers
   This sheet identifies top customers based on total sales.
   Excel functions used:

* INDEX()
* MATCH()
* FILTER()

Top 10 customers are highlighted using conditional formatting.

---

4. What_If_Analysis
   This sheet shows the impact of changing discount values on total profit.
   Excel features used:

* What-If Analysis
* Data Table / Scenario Manager

---

5. Regression_Analysis
   Linear regression is performed using the Data Analysis Toolpak.
   Analysis performed:

* Profit vs Sales

Outputs included:

* R Square
* Coefficients
* Regression summary for interpretation

---

6. Descriptive_Statistics
   This sheet contains descriptive statistics generated using the Analysis Toolpak.
   Includes:

* Mean
* Median
* Standard Deviation
* Minimum
* Maximum

---

7. Pivot_Analysis
   This sheet contains pivot tables used for analysis and dashboard creation.
   Pivot tables created for:

* Total Sales by Region
* Total Sales by Product Category

---

8. Charts
   This sheet contains visual representations of KPIs.
   Charts included:

* Bar Chart: Sales by Region
* Line Chart: Monthly Sales Trend
* Pie Chart: Product Category Contribution

Monthly sales growth includes up and down arrows using conditional formatting.

---

9. Dashboard
   This sheet contains an interactive Excel dashboard.
   Features included:

* Key KPIs (Total Sales, Total Profit, Total Customers)
* Pivot Charts
* Slicers for Region and Product Category

The dashboard is designed for management-level decision making.
